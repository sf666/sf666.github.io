NextCP/2
This is a web-based UPnP control point. Most control point applications run on a single device. This control point allows simultanious access from any device with a current Chrome based web-browser.
User Interface
Content Directory Browsing
Before browsing your media library select a media server from the I/O settings, or define a default media server in the App Setting Folders, playlists and music items are displayed in separate sections.
General content browsing
Depending on the tile type an overlay with symbols appear to
  • step into a folder or playlist
  • delete current playlist and play all music items from this playlist
  • add all music items to current playlist
Album browsing
Visiting albums will display more options.
  • Pressing play will clear the current playlist, add all tracks and start playback.
  • Pressing shuffle will clear the current playlist, add all tracks and start shuffle playback.
  • Pressing add tracks will add all tracks to the current playlist.
Clicking on a title will play the selected track, not adding this item to the current playlist.
Playlist
Displays the current playlist. The active song is highlighted. nextCP/2 supports native OpenHome playlists, as well as control point based playlist for regular UPnP devices.
Click on any music item to listen to it - playback will continue after the song ends.
Online Radio
Click on the play symbol to play an online resource or radio station. Track and album elements are updates if supported by the media renderer device.
Radio
Radio stations confiigured by nextCP/2 configuration file.
Device Radio Station
Radio stations provided by the selected media renderer.
Settings
Client profiles
Save and select a profile for default media server and media renderer. Those defaults apply to the browser which makes the configuration. Those devices will be automatically selected - if available - after entering the application.
Media renderer configuration
List af discovered media renderer. If enabled switch is set and the device is online, the media renderer will be available as a select option in the output drop down. If the media renderer should be controlled by another path but UPnP, select a device driver and enter a configuration string.
Features
Basically the features below are implemented. There might be some usage issues. Feel free to leave a ticket.
  • Browse Content Directory Server
  • UPnP Playlist support
  • Openhome Playlist support
  • play web radio
  • play OpenHome radio
Compatibility
Browser
This software is tested with the current Chrome browser on desktop and mobile devices, but should also work with chromium based browsers like Safari.
Media Server
This software has been tested with
